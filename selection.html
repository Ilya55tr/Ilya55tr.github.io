8 нейросетей для учебы, доступных в России 🇷🇺

Нейросети уже помогают в учебе, но из России бывает сложно зарегистрировать аккаунт или оплатить подписку. 💻 Мы собрали сервисы, которые пригодятся, чтобы проверять факты, создавать сайты, учить языки и разбирать школьные задачи. 📚 Большая часть доступна только на английском. 🌍

Сharacter AI

Языки: английский, русский, испанский
Формат: сайт, приложения для iOS и Android

Текстовый чат-бот, который имитирует разных персонажей. Среди них есть «репетиторы» по математике или английскому, наставники по креативности или программированию. Еще сервис пригодится для тренировки навыков письма

DeepL

Языки: английский и еще 31 язык
Формат: сайт, приложения для iOS и Android, расширение для Chrome

Переводчик, который улавливает языковые нюансы. Переводит не только фрагменты текста, но и файлы, например презентации. Для английского и немецкого уточняет формулировки и меняет тональность письма

TeleportHQ

Язык: английский
Формат: сайт

Сервис для создания интерфейсов и сайтов. Если написать несколько промптов, TeleportHQ сгенерирует рабочий макет и напишет код страницы. Бесплатная версия ограничена тремя проектами

Glasp

Язык: английский
Формат: сайт, расширение для Safari и Chrome

Соцсеть для обмена информацией. Расширение для браузера транскрибирует видео c YouTube и других платформ, выделяет главное и сохраняет в заметки. Учтите, что все материалы попадают в публичный доступ

Perplexity

Язык: английский, но задавать вопросы можно на русском
Формат: сайт, приложение для iOS, расширение для Chrome

Чат-бот ищет информацию, делает саммари из прочитанного и генерирует текст по запросу. Если зарегистрироваться, появится помощник Copilot на базе GPT⁠-⁠4, но количество запросов к нему ограниченно

Yandex GPT

Язык: Русский
Формат: сайт

Yandex GPT — это модель искусственного интеллекта, разработанная для генерации текста на основе заданного запроса. Она может быть полезна для создания учебных материалов, генерации идей и улучшения качества письменных работ.

GigaChat

Языки:Русский и другие языки в зависимости от настроек сервиса.
Формат: сайт

GigaChat — это сервис на базе искусственного интеллекта, который позволяет вести диалог с нейросетью, получать ответы на вопросы и генерировать тексты. Он может быть использован для подготовки к занятиям, поиска информации и развития навыков общения на иностранном языке.

Notion

Язык: русский, но интерфейс на английском
Формат: сайт, приложения для Windows, macOS, Android и iOS

Сервис для продуктивности, в который встроен ИИ⁠-⁠помощник. Может сгенерировать письмо или статью, ответить на вопрос, исправить ошибки или перевести текст. Есть бесплатные запросы для теста, потом сервис попросит купить подписку
